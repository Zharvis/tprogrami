## Agent skills

### Issue tracker

Issues are tracked using GitHub issues via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage roles map directly to standard labels (e.g. `needs-triage`, `ready-for-agent`). See `docs/agents/triage-labels.md`.

### Domain docs

Uses a single-context layout with a global `CONTEXT.md` and `docs/adr/`. See `docs/agents/domain.md`.
